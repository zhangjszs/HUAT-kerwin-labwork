Qt 安装指南 —— 环境变量配置

配置目的

确保在命令行中可以直接访问 Qt 的工具链：

  • qmake - Qt项目构建工具
  • mingw32-make - MinGW版本的Make工具
  • g++ - C++编译器

Windows 配置

需要添加两个路径到系统的 Path 环境变量：

1. Qt 可执行文件目录：

<Qt安装目录>\<版本号>\mingw_XX\bin

例如：C:\Qt\6.5.3\mingw_64\bin

2. MinGW 编译器目录：

<Qt安装目录>\Tools\mingwXXX_XX\bin

例如：C:\Qt\Tools\mingw1120_64\bin

注意： 请根据实际安装路径和版本号进行修改

Linux/macOS 配置

通常安装脚本会自动处理，或者手动修改配置文件：

Linux: 在 ~/.bashrc 或 ~/.zshrc 添加：

export PATH="$HOME/Qt/版本号/gcc_64/bin:$PATH"

macOS: 在 ~/.zshrc 添加：

export PATH="$HOME/Qt/版本号/macos/bin:$PATH"
Windows 环境变量配置
点击"新建"添加Qt和MinGW的bin目录路径

验证配置

安装完成后，打开命令行/终端，输入以下命令检验：

命令提示符

C:\Users\user> qmake -v

QMake version 3.1
Using Qt version 6.5.3 in C:/Qt/6.5.3/mingw_64/lib

C:\Users\user> g++ --version

g++ (GCC) 11.2.0
Copyright (C) 2021 Free Software Foundation, Inc.

成功指标： 如果能显示版本信息，则表示环境变量配置成功！

这意味着您可以在任何目录下使用Qt和MinGW的命令行工具。
