2. Qt 安装指南 —— 安装步骤

Qt 安装指南 —— 安装步骤 (Windows 示例)

1 运行 Qt Online Installer

双击下载的安装程序执行文件

2 登录/注册 Qt Account

即使是开源版也需要注册账号

3 同意许可协议

阅读并接受开源或商业许可条款

4 选择安装路径

建议路径不要有中文或空格

5 选择组件 (关键步骤)

Qt 版本

选择最新的 LTS Desktop MinGW 版本

开发工具 (Developer and Designer Tools)

  • Qt Creator IDE — 必选
  • MinGW — Windows 初学者推荐
  • (可选) MSVC — 如已安装 Visual Studio
  • CMake — 建议勾选

其他模块

根据需要选择，初学可以只选核心模块

6 开始安装

根据网络速度和选择组件，可能需要较长时间

Qt Installer - Component Selection
组件选择是安装过程中最重要的步骤

其他操作系统安装差异

Linux

主要区别在于编译器选择 (GCC)，可能需要安装额外依赖

sudo apt-get install build-essential libgl1-mesa-dev (Ubuntu)

macOS

需安装 Xcode 或 Command Line Tools，使用 Clang 编译器

xcode-select --install

无论哪个系统，组件选择原则是相同的：Qt版本 + Qt Creator + 适合系统的编译器

如遇下载慢的情况，可尝试：(1) 使用代理工具加速；(2) 在校园网环境下载；(3) 使用离线安装包
