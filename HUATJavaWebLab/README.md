# HUAT JavaWeb实验

本项目包含JavaWeb基础实验，使用Servlet、JSP等技术。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **IDE**: IntelliJ IDEA Ultimate 2024.4
- **JDK**: JDK 21
- **服务器**: Tomcat 10.1.19
- **数据库**: MySQL 8.0
- **构建工具**: Maven 3.9 (阿里云镜像)
- **规范**: Jakarta EE 10

## 依赖安装

```bash
# 安装 JDK 21
# 下载并安装 JDK 21

# 安装 Tomcat
# 下载 Tomcat 10.1.19 并配置

# 安装 MySQL
# 下载 MySQL 8.0

# 配置 Maven 阿里云镜像
# 编辑 ~/.m2/settings.xml
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/HUATJavaWebLab
   ```

2. 导入项目：
   - 在 IntelliJ IDEA 中打开 WebLab1 等目录

3. 配置数据库：
   ```sql
   CREATE DATABASE webapp;
   ```

4. 运行项目：
   - 配置 Tomcat
   - 启动服务器

## 目录结构

- `WebLab1/`：基础Servlet实验
- `WebLab2/`：JSP实验
- `WebLab3/`：数据库集成
- `webLab4/`：MVC模式
- `WebLab5/`：综合实验
