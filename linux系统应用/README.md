# Linux系统应用

本项目包含Linux系统应用课程的实验代码和资料。

## 环境要求

- **操作系统**: Linux (Ubuntu/Debian)
- **Shell**: Bash
- **工具**: GCC, Make

## 依赖安装

```bash
# 更新系统
sudo apt update

# 安装开发工具
sudo apt install build-essential
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/linux系统应用
   ```

2. 运行脚本：
   ```bash
   chmod +x script.sh
   ./script.sh
   ```
