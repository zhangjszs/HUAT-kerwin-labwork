# JavaWeb课程设计

简单博客系统，使用Spring Boot实现的Web应用。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **JDK**: Java 17+
- **构建工具**: Maven 3.6+

## 依赖安装

```bash
# 安装 JDK
# 下载并安装 JDK 17

# 安装 Maven
# Windows: 下载安装包
# Linux: sudo apt install maven
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/javaweb课设
   ```

2. 构建运行：
   ```bash
   cd ForestBlog-master
   mvn clean install
   mvn spring-boot:run
   ```

3. 访问应用：
   打开浏览器访问 http://localhost:8080

## 目录结构

- `ForestBlog-master/`：博客系统源码
  - `src/`：源代码
  - `pom.xml`：Maven配置

## 功能说明

- 用户注册登录
- 文章发布管理
- 评论系统
- 权限管理
