# Titanic 生存预测分析

基于经典 Titanic 数据集的二分类预测项目，完整展示数据科学建模全流程。

## 技术栈

- Python 3.10+
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
- xgboost

## 运行方式

```bash
pip install pandas numpy matplotlib seaborn scikit-learn xgboost jupyter
jupyter notebook titanic_analysis.ipynb
```

## 文件说明

- `titanic_analysis.ipynb` — 完整分析 Notebook（问题定义 → EDA → 特征工程 → 多模型对比 → 模型解释 → 结论）
