# C语言课程设计

车辆索赔管理系统，使用C语言实现的文件管理系统。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **编译器**: GCC 或 MinGW
- **构建工具**: CMake 3.10+

## 依赖安装

```bash
# 安装 CMake
# Windows: 下载安装包
# Linux: sudo apt install cmake
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/C语言课设
   ```

2. 构建项目：
   ```bash
   mkdir build
   cd build
   cmake ..
   cmake --build .
   ```

3. 运行程序：
   ```bash
   ./VehicleClaimManagement
   ```

## 目录结构

- `vehicle_claim.c`：主程序
- `vehicle_claim.h`：头文件
- `CMakeLists.txt`：构建配置
- `output/`：输出文件
- `新图片/`：图片资源

## 功能说明

- 车辆信息管理
- 索赔记录管理
- 文件持久化存储
- 多条件查询
