# 编译原理课程实验

本项目包含编译原理课程的实验代码，包括词法分析、语法分析等编译器构造技术。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **编译器**: GCC 11.4.0 或更高版本
- **开发环境**: Visual Studio Code 或 CLion

## 依赖安装

```bash
# 无需额外依赖，使用标准库
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/编译原理
   ```

2. 编译运行：
   ```bash
   cd Code
   g++ lexer.cpp -o lexer
   ./lexer
   ```

## 目录结构

- `Code/`：核心代码实现
- `experiment1/`：词法分析实验
- `experiment2/`：语法分析实验
