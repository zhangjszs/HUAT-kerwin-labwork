# HUAT Java Homework

本项目包含Java课程作业集合。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **JDK**: Java 8+
- **IDE**: IntelliJ IDEA

## 依赖安装

```bash
# 安装 JDK
# 下载并安装 JDK 8+
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/schoolJavaHomework
   ```

2. 导入项目：
   - 在 IntelliJ IDEA 中打开项目

3. 编译运行：
   ```bash
   javac src/*.java
   java -cp src Main
   ```

## 目录结构

- `src/`：源代码
- `out/`：编译输出
