# Java课程设计

本项目包含Java课程设计的游戏项目，使用Swing框架。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **JDK**: Java 8+
- **IDE**: IntelliJ IDEA 或 Eclipse

## 依赖安装

```bash
# 安装 JDK
# 下载并安装 JDK 8+
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/java课程设计
   ```

2. 解压项目：
   ```bash
   unzip 大鱼吃小鱼源码.zip
   ```

3. 编译运行：
   ```bash
   javac *.java
   java Main
   ```

## 目录结构

- `大鱼吃小鱼源码.zip`：大鱼吃小鱼游戏
- `项目2贪食蛇.zip`：贪食蛇游戏
- `项目3吞食鱼.zip`：吞食鱼游戏
- `项目4超级玛丽.zip`：超级玛丽游戏
