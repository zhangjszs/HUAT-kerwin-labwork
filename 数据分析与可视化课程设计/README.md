# 数据分析与可视化课程设计

本项目包含数据分析与可视化课程设计，包括竞赛项目和课程作业。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **Python**: 3.10+
- **Jupyter**: 支持Notebook

## 依赖安装

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/数据分析与可视化课程设计
   ```

2. 启动Jupyter：
   ```bash
   jupyter notebook
   ```

3. 打开并运行notebook文件。

## 目录结构

- `anylize.ipynb`：数据分析notebook
- `taobao.ipynb`：淘宝数据分析
- `boos/`：BOSS直聘数据
- `报告模板/`：报告模板
- `比赛项目.md`：竞赛项目说明
