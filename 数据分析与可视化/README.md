# 数据分析与可视化课程实验

本项目包含数据分析与可视化的实验代码，使用 Python 和 Jupyter Notebook 进行数据处理和可视化。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **Python**: 3.10 或更高版本
- **Jupyter**: 支持 Notebook
- **内存**: 至少 4GB RAM

## 依赖安装

```bash
# 安装 Python 环境
pip install -r requirements.txt

# 或使用 conda
conda env create -f environment.yml
```

### 主要依赖
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- jupyter

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/数据分析与可视化
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 启动 Jupyter：
   ```bash
   jupyter notebook
   ```

4. 打开实验 notebook 并运行。

## 目录结构

- `experiment1/`：基础数据处理
- `experiment2/`：数据可视化
- ... 其他实验
- `requirements.txt`：依赖列表

## 实验说明

每个实验包含：
- Jupyter Notebook (.ipynb)
- 数据文件
- 实验报告
