# 操作系统课程实验

本目录包含操作系统课程的实验代码，涵盖进程管理、内存管理、文件系统等核心内容。

| 实验编号 | 实验名称 | 目录 | 主要内容 | 技术栈 |
|-----------|---------|------|---------|---------|
| 实验一 | 进程管理 | [labwork1/](./labwork1/) | 进程创建、同步、通信 | C++ |
| 实验二 | 内存管理 | [labwork2/](./labwork2/) | 内存分配、回收算法 | C |
| 实验三 | 进程同步 | [labwork3/](./labwork3/) | 信号量、互斥锁 | C++ |
| 实验四 | 文件系统 | [labwork4/](./labwork4/) | 文件操作、目录管理 | C++ |
| 实验五 | 设备管理 | [labwork5/](./labwork5/) | 设备驱动、I/O控制 | C++ |
| 实验六 | 死锁与银行家问题 | [labwork6/](./labwork6/) | 死锁检测、避免算法 | C++ |

## 🚀 快速开始

### 编译与运行

```bash
# 进入实验目录
cd <实验目录>

# 编译（以实验一为例）
cd labwork1
g++ labwork1.1.cpp -o labwork1.1

# 运行
./labwork1.1
```

## 📊 实验亮点

- ✨ **系统调用**：使用 Linux 系统调用实现进程管理
- 🎯 **算法实现**：内存分配算法（首次适应、最佳适应等）
- 🔍 **同步机制**：信号量、互斥锁的正确使用
- 📝 **详细注释**：便于理解操作系统原理

## 💡 学习建议

1. **理解进程概念**：进程、线程、进程间通信是核心概念
2. **掌握内存管理**：理解虚拟内存、分页、分段机制
3. **实践文件系统**：通过实验理解文件的组织和管理
4. **注意并发问题**：死锁、饥饿、竞争条件的产生和避免
