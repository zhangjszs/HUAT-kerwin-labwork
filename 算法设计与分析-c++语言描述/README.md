# 算法设计与分析课程实验

本项目包含算法设计与分析课程的实验代码，使用C++实现各种经典算法。

## 环境要求

- **操作系统**: Windows/Linux/macOS
- **编译器**: GCC 11.4.0 或更高版本，支持C++11
- **IDE**: Visual Studio Code, CLion 等

## 依赖安装

```bash
# 无需额外依赖，使用标准库
```

## 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/zhangjszs/HUAT-kerwin-labwork.git
   cd HUAT-kerwin-labwork/算法设计与分析-c++语言描述
   ```

2. 编译运行：
   ```bash
   cd experiment1
   g++ main.cpp -o algorithm_demo
   ./algorithm_demo
   ```
